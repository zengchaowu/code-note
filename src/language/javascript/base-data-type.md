# 数据类型

## 基本数据类型及其用途

| 基本类型  | 用途                                     |
| --------- | ---------------------------------------- |
| boolean   | 存储 true 或者 false                     |
| null      | 空值                                     |
| undefined | 未初始化                                 |
| number    | 存储整数或者浮点数，可能会出现精度问题。 |
| bigint    | 用来安全地存储和操作大整数               |
| string    | 存储文本                                 |
| symbol    | 唯一且不可改变的数据类型                 |
